### Usage
Start off by trying out a simple get request
```
gurl <URL>
```
Feeling comfortable? Try adding data to the request

*template*
```
--d, --data

gurl <URL> --data NAME1:DATA1 NAME2:DATA2
```

*example*
```
gurl catpictures.com/api/random --data length:10
```
